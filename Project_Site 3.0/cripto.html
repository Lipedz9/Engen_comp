Componentes Básicos da Criptografia

Algoritmo Criptográfico:

É o conjunto de regras e operações matemáticas utilizadas para criptografar e descriptografar dados. Existem diversos algoritmos, como AES (Advanced Encryption Standard), RSA (Rivest-Shamir-Adleman), entre outros, cada um com suas características e níveis de segurança.

Chave Criptográfica:

É um valor secreto utilizado pelo algoritmo criptográfico para transformar os dados originais em dados criptografados (e vice-versa). Existem chaves simétricas (mesma chave para criptografar e descriptografar) e chaves assimétricas (chave pública e chave privada).

Protocolos de Criptografia:

São conjuntos de regras que especificam como os algoritmos criptográficos e as chaves são usados para proteger os dados em diferentes cenários, como transmissão de dados pela internet (por exemplo, HTTPS, TLS/SSL).

Tipos de Criptografia:

  • Criptografia Simétrica: Utiliza a mesma chave para criptografar e descriptografar os dados. É mais rápida e eficiente, ideal para grandes volumes de dados. Exemplos incluem AES e DES.
  • Criptografia Assimétrica: Utiliza um par de chaves: uma pública e outra privada. A chave pública é usada para criptografar dados ou verificar assinaturas digitais, enquanto a chave privada é usada para descriptografar dados ou assinar digitalmente. Exemplos incluem RSA e ECC.

Funções de Hash Criptográfico:

São algoritmos que transformam dados em uma sequência de bytes fixa (hash) de tamanho definido. São usadas principalmente para garantir integridade dos dados e verificar a autenticidade dos dados. Exemplos incluem SHA-256 e MD5 (não recomendado para segurança devido a vulnerabilidades conhecidas).

Aplicações da Criptografia:

  • Comunicações Seguras: Garantindo que dados transmitidos pela internet sejam confidenciais e não possam ser lidos por terceiros não autorizados.
  • Armazenamento Seguro: Protegendo dados armazenados em dispositivos de armazenamento físicos ou em nuvem contra acesso não autorizado.
  • Autenticação e Assinaturas Digitais: Verificação da autenticidade de dados e identidade de usuários através de chaves criptográficas.
